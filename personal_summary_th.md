# สรุปผลการทดลองด้วยความเข้าใจของตัวเอง

---

## 1. Research Question

เป็นการทดลองใช้ Model AI Computer Vision 4 ตัว ในการเทรนโดยใช้รูปแบบ Modality WLI เพียงอย่างเดียว เพื่อศึกษาว่า Architecture ที่เลือกใช้ส่งผลต่อความสามารถในการตรวจจับกับกล้องประเภทอื่นๆ ยังไงบ้าง

---

## 2. EDA — การสำรวจข้อมูลเบื้องต้น

Dataset นี้ไม่สมดุลอย่างมาก เพราะ WLI มีมากถึง 3,588 รูป ในขณะที่กล้องประเภทอื่นมีเพียง 60-146 รูปต่อประเภทเท่านั้น ความไม่สมดุลนี้คือความท้าทายหลักของงานครั้งนี้ และนำไปสู่การออกแบบการทดลองแบบ Cross-Modal เพื่อทดสอบว่า Model ที่เทรนจาก WLI จะ generalize ไปยังกล้องชนิดอื่นได้ดีแค่ไหน

---

## 3. Pipeline

โครงสร้าง Pipeline หลักของการทำ Object Detection คือการสร้าง Bounding Box เพื่อจับความผิดปกติภายในกรอบที่กำหนด โดยดึงข้อมูลจาก Dataset แปลง Mask เป็น Bounding Box แล้วนำไปเทรนทั้ง 4 Model เพื่อนำมาเปรียบเทียบและประเมินผลข้าม Modality แต่ละแบบ

---

## 4. การคัดเลือก Model

เลือกใช้ 4 Architectures ที่มีขนาดต่างกัน เพื่อวัดว่า Parameter ที่มากกว่าจะส่งผลให้ความสามารถในการ Generalize ดีกว่าหรือไม่ และมีนัยสำคัญมากน้อยแค่ไหน

---

## 5. ผลลัพธ์

**YOLO26s** มีค่าความแม่นยำต่ำมากใน NBI และ BLI (76-79%) เพราะกล้องประเภทนี้ใช้แสง Narrow-Band สีน้ำเงิน ส่งผลให้สีของภาพผิดเพี้ยนไปจากความเป็นจริงอย่างมาก ต่างจาก WLI (98%) ที่ใช้แสงขาวทำให้เห็นเนื้อเยื่อในสีใกล้เคียงความจริงมากที่สุด เปรียบเหมือนแค่ส่องไฟให้สว่างขึ้นโดยไม่เปลี่ยนสี

**YOLOv11s** คงที่ทุก Modality ที่ 94%+ แสดงว่า Architecture ใหม่กว่า generalize ได้ดีกว่าอย่างมีนัยสำคัญ

---

## 6. Key Finding

ขนาดของ Model ไม่ได้การันตีว่า Model จะเข้าใจจริงๆ YOLO26s มีขนาดใกล้เคียง YOLOv11s แต่ผลต่างกันมาก เพราะ YOLOv11s เข้าใจ Shape ของติ่งเนื้อได้ดีกว่า แทนที่จะพึ่งพาสีและ Texture ของกล้องชนิดเดิม

---

## 7. Error Analysis

YOLO26s over-rely สีและ Texture ของ WLI มากเกินไป พอแสงเปลี่ยนไปเป็น Narrow-Band Model จึงสับสนและคลาดเคลื่อนอย่างมาก ช่องว่างที่ยังเหลืออยู่ใน YOLOv11s บน NBI/BLI นั้นไม่ใช่ความผิดพลาดของ Model แต่เป็น Domain Gap ที่เกิดจากความต่างของแสงจริงๆ ซึ่งต้องแก้ด้วยการเพิ่ม Training Data จาก Modality เหล่านั้น

---

## 8. ข้อจำกัดและสิ่งที่ต้องทำต่อ

จากการทดลอง HSV Ablation พบว่า HSV=0.7 ให้ผลดีกว่า HSV=0 ในเกือบทุก Modality ยกเว้น BLI ที่ HSV=0 ทำได้ดีกว่า ซึ่งน่าสนใจมากเพราะ BLI คือ Modality ที่แตกต่างจาก WLI มากที่สุด

ข้อจำกัดของโปรเจกต์นี้คือ:
- เทรนด้วย WLI เพียงอย่างเดียว ถ้าเพิ่ม Modality อื่นเข้าไปในการเทรนน่าจะได้ผลดีขึ้น
- ไม่ได้ Tune Hyperparameter อย่างเป็นระบบ — ครั้งนี้ทำได้แค่ HSV ablation เพียงตัวเดียว
- Test set ของ Minority Modality มีน้อยมาก (7-14 รูป) ทำให้ผลอาจไม่เสถียรพอที่จะสรุปได้อย่างมั่นใจ 100%
- ยังไม่ได้ทำ Per-image Error Analysis เพื่อดูว่า Model พลาดรูปแบบไหนบ้าง
